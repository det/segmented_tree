Function template operator<

boost::operator<

Synopsis

// In header: <boost/segmented_tree_seq.hpp>


template<typename T, typename Alloc, size_t... Args> 
  bool operator<(segmented_tree_seq< T, Alloc, Args... > & lhs, 
                 segmented_tree_seq< T, Alloc, Args... > & rhs);

Description

Returns. True if the first sequence is lexicographically less than the second. False otherwise.

Complexity. Linear in size().

Iterator invalidation. Iterators are not invalidated.

Exception safety. Strong.
